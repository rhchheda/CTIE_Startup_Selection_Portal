# KLE CTIE Incubation Portal

**Applications, jury review, and staff decisions for KLE-CTIE's startup incubation cohorts — in one place.**

Startups apply once during an open cohort window, an assigned jury scores every submission against
a fixed rubric, and CTIE staff track the pipeline through to a final decision (Shortlisted / Selected
/ Not Selected / On Hold) — with automatic email updates at every stage instead of a paper trail split
across inboxes and spreadsheets.

## What it does

- **Applicants** sign in (Google account or a passwordless one-time email code — any email works,
  no account needed in advance) and submit one application per open cohort through a 6-step wizard:
  self info, startup info, product & revenue, financials & stage, business plan, and supporting
  documents. They can track and edit their submission right up until a decision is recorded.
- **Jury members** only ever see applications CTIE staff has explicitly assigned to them, score each
  one against a fixed 6-criterion rubric, and leave a Select / Reject / Hold recommendation.
- **CTIE Staff/Admin** open and close cohorts, assign jurors, and record the final decision per
  application. A Shortlisted decision automatically creates a Google Calendar event with a Meet link
  and invites the applicant and every assigned juror.
- Every privileged action is re-checked against the applicant/jury/staff role on the server side,
  every time — the client never gets to assert who it is.

## About this repository

This is a Google Apps Script project. The application itself (server logic, HTML, and configuration)
is maintained privately within Google Apps Script and is **not** published here — this repository
only carries the public-facing overview above and the presentation in [`docs/index.html`](docs/index.html).
KLE CTIE team members with a reason to work on the source should request access directly rather than
looking for it in this repo.
